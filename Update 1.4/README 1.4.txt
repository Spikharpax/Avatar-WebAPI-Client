************************
Mise à jour version 1.4
************************

Ce patch n'est à installer que si vous avez une version 1.3
Pour une installation complète, le package d'installation AvatarWebAPIClient.7z contient déjà ce patch.

Mise à jour:
	- Optimisation de la communication entre A.V.A.T.A.R et Google Chrome pour une meilleure stabilité du dialogue 
	- Optimisation du redémarrage automatique après une perte de connection
	- Mise à jour de la documentation

Procédure d'installation :
	- Copiez le contenu de ce répertoire dans le répertoire <CLIENT>/
		- Validez les remplacements de fichiers
	- IMPORTANT: Changez la propriété "version" en 1.4 dans le fichier de propriétés <CLIENT>/resources/app/Avatar.config
