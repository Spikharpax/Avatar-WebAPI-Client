Installation

Après avoir suivi la procédure d'installation du client depuis le projet github, installez ensuite les applications connexes:


    1.  Si besoin, téléchargez et installez Microsoft .NET Framework 4.5
          • Dans un environnement récent, ce framework est déjà installé. Vous pouvez le vérifier dans les programmes installés ou sinon dans tous les cas, le programme d'installation vous signifiera si vous n'avez pas besoin de l'installer.
    1. Téléchargez et installez la version x86 de Microsoft Speech Plateform - Runtime (Version 11)
    2. Accessoirement (non obligatoire), vous pouvez aussi téléchargez une voix sur Microsoft Speech Plateform - Runtime language (version 11)
      • Pour le Français, vous pouvez sélectionner la voix MSSpeech_TTS_fr-FR_Hortense.msi
      • Au chapitre synthèse vocale, vous avez aussi la possibilité d'installer d'autres voix plus intéressantes.



A noter: Kinect pour Windows V1 ou V2 ne fonctionne pas avec ce client.
