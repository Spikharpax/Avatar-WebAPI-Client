************************
Mise à jour version 1.1
************************

Ce patch n'est à installer que si vous avez une version 1.0
Pour une installation complète, le package d'installation AvatarWebAPIClient.7z contient déjà ce patch.

Mise à jour:
	- Nouvelle façon de dicter des règles, vous pouvez maintenant dicter plusieurs règles dans la même phrase
	- Possibilité de tester les mots-clés et d'afficher le résultat dans la console Chrome
	- Nouvelle règle pour remettre l'écoute si celle_ci est bloquée

Procédure d'installation :
	- Copiez les répertoires "resources" et "documentation" dans <CLIENT>/
	- Validez les remplacements fichiers
	- N'oubliez pas de changer la propriété "version" en 1.1 dans le fichier de propriétés Avatar.config
